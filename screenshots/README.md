# Screenshots Directory

## 📸 产品效果图说明

本目录用于存放产品功能截图，用于在 README 中展示。

### 需要的截图文件：

1. **daily-report-entry.png** - 日报录入界面
   - 展示日历视图、日报输入框、TODO Tips 面板
   - 确保显示已录入日期的绿色标记和周末红色字体

2. **weekly-report-generator.png** - 周报生成界面
   - 展示从日报导入功能、日期范围选择
   - 显示生成的周报内容

3. **weekly-report-query.png** - 周报查询界面
   - 展示历史周报查询列表
   - 显示编辑/删除功能

4. **okr-generator.png** - OKR 管理界面
   - 展示 OKR 生成功能
   - 显示目标和关键结果的结构化展示

### 截图要求：

- **分辨率**：建议 1920x1080 或更高
- **格式**：PNG 格式（支持透明背景）
- **字体**：确保截图中的中文字体清晰可见，无乱码
- **内容**：包含真实的功能演示数据（非敏感信息）
- **标注**：可适当添加标注说明关键功能点

### 如何截图：

1. 启动应用（使用 `start_services.bat`）
2. 访问 http://localhost:5002
3. 依次进入各个功能页面
4. 使用 Windows 截图工具（Win+Shift+S）或浏览器截图功能
5. 保存为对应文件名到本目录

### 注意事项：

- ✅ 确保浏览器缩放比例为 100%
- ✅ 关闭浏览器开发者工具
- ✅ 确保页面完整显示，无滚动条
- ✅ 使用一致的浏览器和主题
- ✅ 截图前检查是否有测试数据展示
